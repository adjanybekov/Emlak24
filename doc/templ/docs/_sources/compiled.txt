.. _compiled_version:

Compiled version (recommended)
==============================
The compiled version is located in the ``template/`` directory and is the one that you're most probably going to use.
No setup whatsoever is required.

Structure
---------

* ``template/assets/js/`` - JS files with libraries, core functionality and examples (see html comments for details).
* ``template/assets/fonts/`` - Font icons
* ``template/assets/img/`` - Image assets that are used in the theme
* ``template/*.html`` - HTML templates
